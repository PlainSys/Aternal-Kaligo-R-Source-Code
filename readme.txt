                   ❚█══Aternal Kaligo (Saweba) Remastered══█❚
                                The First Mission
                                Source Code Release

This is the source code of the remastered version of Aternal Kaligo (then known
as Saweba which is translated from the Arabic word of "difficult" or "hard"
which dates back to July 8th, 2024 before being updated several more times and
being compressed on August 12th, 2025. The reason this version was created was
because the original Saweba 1 from 2021 is now lost media alongside some other
Godot projects made around 2021-2022. The remastered version was made using an
early backup of the game while incorporating some original elements. Due to
this... this game is pretty shit and was kinda made to keep the shitty feel.

Some things in this game/version are also broken/were never addressed such as
certain score giving bonus lives and the extra stage being unlockable. One
day, I might remake (or demake) this game to make it more polished and be in
line with its sequel. Anyways, its funny when I showcased some friends Saweber
2 and they go "okay but where's Saweba 1?" and well, here it is. Another question
before I go into how to open this is the switch from Saweba to Aternal Kaligo.
Saweba 1 (and even 2) were goofy games without any serious stuff to it. I'd
honestly go far as to say it had that jokey feel. In recent times however,
the game has been getting some deep stuff/lore around it. I thought now was a
good time then ever to rebrand this game... franchise... since its barely popular
and is only known within friend groups. I wanted that Egypt feel to the name still
since... the game is based around some Egypt box. I wanted a name that fit in
line with something like "Killer Instinct", some edgy stuff that relates to the game.
I looked up some Latin shit cuz that always goes hard and I got Caligo I think which
means darkness. I stylized eternal and caligo and bam, funny name.

How to open project:
This was made with Godot 4 (idk which specific version but I think I downloaded Godot as
recently as 2023-early 2024 before uninstalling. Anyways download the folder, open it in
Godot, you should have the contents of the folder.

What can I do with this?
You can modify the game to your desire. My only rules are to not sell them, always credit
meh (Plain or PlainSys) as the creator of the game, and don't do some freaky shit/mods...
weirdo.

Anyways, you can make custom levels with this, custom bosses, a whole new campaign ig...
and prob custom playable characters?
